🌦️ Weather Analytics Dashboard (Power BI + Weather API)

📌 Description :

A dynamic and interactive "Weather Analytics Dashboard" built using "Power BI" with "live weather and forecast data" fetched from WeatherAPI. It provides real-time weather updates, 7-day forecast trends, air-quality insights, and city-wise comparisons to support analysis and decision-making.


🛠️ Tech Stack :

* Power BI Desktop
* Power Query (M Code)
* DAX (Data Analysis Expressions)
* WeatherAPI (Free Tier)
* JSON Data Processing
* API Integration using Web Connector


🌐 Data Source :

* WeatherAPI – Current Weather Endpoint
* WeatherAPI – Forecast (7 Days) Endpoint

Data extracted using Power BI’s "Web Connector", then transformed and modeled into:

* "Current_Data" table
* "Forecast_Data" table (daily + hourly)
* "Master table" merging all city data

All unnecessary tables were cleaned or hidden for efficient modeling.


✨ Features / Highlights :

 1. Business Problem

Organizations, travel planners, and operations teams often need "real-time weather insights", multi-city comparisons, and air-quality information to make smart decisions. Manual tracking is unreliable and not scalable.

 2. Goal of the Dashboard

* Provide "live", accurate weather information for selected cities
* Allow users to analyze "current conditions", "future forecasts", and "air quality levels"
* Present weather risks (rain, humidity, UV index) clearly
* Enable quick comparison across multiple locations
* Support data-driven planning for operations, travel, logistics, and events

 3. Key Visuals Included

* "Live Weather Card" — Temperature, condition, humidity, pressure
* "7-Day Forecast Chart" — Daily temperature trend with icons
* "Air Quality Index Gaug" — PM levels, SO2, NO2, O3, CO
* "Sunrise & Sunset Timings"
* "Chance of Rain Visualization" — Percentage bars for each day
* "Wind Speed, Visibility, UV Index Cards"
* "City Selector (Multiple Cities)"

 4. Business Impact & Insights

* 📍 "Improved Planning": Helps in scheduling events, travel, and outdoor activities based on weather conditions.
* 🔄 "Risk Reduction": Forecast-based visibility (rain, UV, storms) supports safety decisions.
* 🌫️ "Pollution Monitoring": AQI visuals help sensitive groups avoid harmful outdoor exposure.
* 📈 "Better Operational Decisions": Companies relying on logistics or outdoor work can adjust operations.
* 🔍 "Data Transparency": Users can compare multiple cities instantly with live updates.
* ⚡ "Automated Refresh" ensures the dashboard always displays updated weather.
